# 🪙 FCDex Economy & Games

The official economy engine for the **FCDex Ecosystem**. Built with **TypeScript** and **Discord.js**, this bot manages the server's primary currency: **Pounds (£)**.

## 🚀 Core Features
* **Dynamic Economy:** Earn, save, and spend **Pounds**.
* **High-Risk Systems:** Engage in the `!rob` protocol to build your wealth (or lose it all).
* **Games of Chance:** Coinflip, Dice, and more (coming soon).
* **Futuristic UI:** Clean, glassmorphic embeds designed for the modern Discord experience.

## 📁 Project Structure
* `/src/commands/economy`: Core financial commands (!balance, !pay, !rob).
* `/src/commands/games`: Betting and chance-based mini-games.
* `/src/utils`: The central logic for "Pounds" calculation and database syncing.

## 🛠️ Tech Stack
* **Language:** TypeScript
* **Library:** Discord.js v14
* **Database:** Prisma + LibSQL (Optimized for FCDex infrastructure)
* **Design:** Photopea-crafted assets for futuristic branding.
